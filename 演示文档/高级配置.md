# UserStoryLLM 高级配置指南

## 大语言模型配置

### 切换 LLM 提供商

#### 使用 DeepSeek（默认）

```env
LLM_PROVIDER=deepseek
LLM_MODEL=deepseek-chat
DEEPSEEK_API_KEY=your_api_key
LLM_TEMPERATURE=0.7
LLM_MAX_TOKENS=4000
```

#### 使用 OpenAI

```env
LLM_PROVIDER=openai
LLM_MODEL=gpt-4
OPENAI_API_KEY=your_api_key
LLM_TEMPERATURE=0.7
LLM_MAX_TOKENS=4000
```

#### 使用本地模型（Ollama）

```env
LLM_PROVIDER=ollama
LLM_MODEL=llama2
OLLAMA_BASE_URL=http://localhost:11434
LLM_TEMPERATURE=0.7
```

### 模型参数调优

| 参数 | 范围 | 说明 |
|-----|------|------|
| TEMPERATURE | 0-1 | 越低输出越确定，越高越创意性。建议 0.5-0.8 |
| MAX_TOKENS | 512-8000 | 最大输出长度，根据需求调整 |
| TOP_P | 0-1 | Nucleus sampling，建议保持默认 0.9 |

### 性能优化

```env
# 启用缓存
LLM_ENABLE_CACHE=true
LLM_CACHE_TTL=3600

# 批量处理
LLM_BATCH_SIZE=5
LLM_BATCH_TIMEOUT=30

# 重试策略
LLM_MAX_RETRIES=3
LLM_RETRY_DELAY=2
```

## 知识库定制

### 添加自定义知识模块

编辑 `backend/seed_common_knowledge.py`：

```python
KNOWLEDGE_BASE = {
    "自定义模块名": {
        "aliases": ["别名1", "别名2"],
        "必选要素": ["要素1", "要素2"],
        "前置条件": ["条件1", "条件2"],
        "异常场景": ["异常1", "异常2"],
        "典型任务": ["任务1", "任务2"],
        "安全约束": "安全要求说明",
        "description": "模块描述"
    }
}
```

### 通过 API 添加知识模块

```bash
curl -X POST "http://localhost:8000/api/v1/kb/add_module" \
  -H "Content-Type: application/json" \
  -d '{
    "module_name": "我的模块",
    "category": "自定义分类",
    "aliases": ["模块别名"],
    "required_elements": ["必选要素"],
    "preconditions": ["前置条件"],
    "exception_scenarios": ["异常场景"],
    "typical_tasks": ["典型任务"],
    "security_constraints": "安全约束",
    "description": "模块描述"
  }'
```

### 定期备份知识库

```bash
# 导出知识库
python -c "
from database import SessionLocal
from knowledge_models import KnowledgeModule
import json

db = SessionLocal()
modules = db.query(KnowledgeModule).all()
data = [m.to_dict() for m in modules]
with open('kb_backup.json', 'w', encoding='utf-8') as f:
    json.dump(data, f, ensure_ascii=False, indent=2)
"

# 恢复知识库
python -c "
import json
from database import SessionLocal
from knowledge_models import KnowledgeModule

db = SessionLocal()
with open('kb_backup.json', 'r', encoding='utf-8') as f:
    data = json.load(f)
    for item in data:
        module = KnowledgeModule(**item)
        db.add(module)
db.commit()
"
```

## Docker 生产部署

### 自定义镜像构建

```dockerfile
# backend/Dockerfile
FROM python:3.10-slim

WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

ENV PYTHONUNBUFFERED=1
ENV LLM_TEMPERATURE=0.7

EXPOSE 8000
CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "8000"]
```

### 多阶段构建前端

```dockerfile
# frontend/Dockerfile
FROM node:18-alpine AS build
WORKDIR /app
COPY package*.json ./
RUN npm ci
COPY . .
RUN npm run build

FROM nginx:alpine
COPY --from=build /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```
